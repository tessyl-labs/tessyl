# tessyl

A full-stack Turborepo application written in [Voyd](../voyd).

## Workspace layout

- `apps/*` — deployable Voyd applications
- `packages/*` — shared Voyd packages

The workspace is ready for the Voyd app initializer; no application has been
generated yet.

## Commands

```sh
npm install
npm run dev
npm run build
npm test
npm run lint
npm run typecheck
```
